Saiba mais sobre o Ruby

Ruby é um bot que faz parte te um projeto que busca ensinar rede de computadores para os usuários do Discord. Esse projeto começou inicialmente sendo um Trabalho de Conclusão de Curso, e hoje obtém um resultado agradavél para toda a comunidade.
Todo conteúdo disponibilizado por Ruby foi revisado e aprovado por Lucas Oliveira, professor de rede de computadores do Instituto Federal de São Paulo - Campûs Barretos
